# On-device / Emulator Validation

Последний подтвержденный технический прогон: 28 марта 2026 года.

## Подтверждено
- `Главная` открывается в `PREPARING`
- нижняя навигация работает для `Главная`, `События`, `Чек-листы`, `Контакты`
- карточка ребенка появляется на `Главной` после перехода в `BABY_BORN`
- экран `События` открывается как отдельный top-level экран
- `Чек-листы` и `Контакты` отрисовываются без падений
- `Настройки` открываются и показывают ПДР

## Проверено на эмуляторе
- smoke-навигация
- device scenario tests
- сценарии аналитики схваток

## Ограничение
Глубокая валидация содержимого `Событий` в `BABY_BORN` дополнительно покрыта unit-тестами `EventsProviderTest`, а не только device-smoke, чтобы не завязывать тесты на хрупкие viewport-ожидания.
