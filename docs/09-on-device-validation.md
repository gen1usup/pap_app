# Подтвержденные проверки на устройстве и эмуляторе

## Последний подтвержденный прогон

Дата: 27.03.2026

Стенд:

- эмулятор `emulator-5554`
- debug APK + androidTest APK из текущей ветки

## Что подтверждено

### 1. Navigation shell

Подтверждено:

- bottom navigation содержит только `Главная`, `События`, `Чек-листы`;
- `Журнал` открывается через bell icon в top bar;
- drawer открывается из shell-экранов;
- drawer содержит две группы: `Этапы` и `Сервис`.

### 2. Stage screen

Подтверждено:

- нажатие по этапу в drawer открывает отдельный stage screen;
- screen показывает описание сценария и кнопку активации;
- stage screen не ломает shell-навигацию.

### 3. Главная

Подтверждено:

- без ПДР показывается reminder-карточка с переходом в настройки;
- `Главная -> Чек-листы -> Главная` через bottom nav работает;
- в `AT_HOSPITAL` на главной скрыт shortcut счетчика схваток.

### 4. События и журнал

Подтверждено:

- bell открывает `Журнал`;
- в пустом журнале отображается empty state;
- пустое labor-событие не сохраняется и показывает ошибку;
- в `AT_HOSPITAL` экран `События` показывает `Приехали домой` и скрывает родовые live-tools;
- в `AT_HOME` экран `События` показывает домашние быстрые записи.

### 5. Настройки

Подтверждено:

- экран настроек открывается из drawer;
- блок ПДР доступен;
- inline-валидация неверной даты работает.

### 6. Справка и SOS

Подтверждено:

- `Справка` показывает блок роддома и маршрута;
- `SOS` показывает `Позвонить 112`, тревожные признаки и переход к контактам.

### 7. Аналитика схваток

Подтверждено на seeded-сценариях:

- нерегулярные схватки остаются в режиме `Наблюдаем`;
- умеренно регулярные схватки дают `Готовимся к выезду`;
- выраженно регулярные схватки дают `Пора ехать`.

## Что пока не подтверждено отдельным UI-прогоном

На этом шаге не оформлены как отдельный e2e сценарий:

- полный путь `схватки -> воды -> рождение -> домой` одним непрерывным автотестом;
- CRUD экстренных контактов через полноценный сценарий редактирования.

Эти пункты остаются кандидатами для следующего набора instrumented tests.
