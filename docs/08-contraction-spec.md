# Спецификация экрана «Активные роды»

## Назначение
Экран объединяет в одном месте:
- состояние схваток
- состояние вод
- рекомендацию
- фиксацию рождения

## Показатели
Экран показывает:
- текущую рекомендацию
- длительность активной схватки или статус ожидания
- текущий интервал
- статус вод
- историю завершенных схваток

## Приоритет рекомендации
1. если рождение уже записано, родовой сценарий больше не основной
2. если воды не прозрачные, рекомендация срочная
3. если воды прозрачные, нужен контакт и наблюдение
4. если воды не отходили, используется расчет по схваткам

## Удаление
Пользователь может удалить:
- отдельную схватку
- запись о водах

После удаления:
- расчет интервалов обновляется
- recommendation policy пересчитывается заново

## Доступность
Экран доступен:
- из `Главной`
- из `Событий`
- после рождения история может использоваться повторно, если пользователь вручную вернется в `LABOR`
