# Правила отображения

## Главная
### `PREPARING`
Показываются:
- reminder по ПДР, если она отсутствует
- подготовительные блоки
- чек-листы этапа
- недавние события

### `LABOR`
Приоритет блоков:
1. live-сводка `Активных родов`
2. статус вод
3. быстрый доступ к контактам
4. чек-лист этапа
5. последние события

### `BABY_BORN`
Порядок:
1. карточка ребенка
2. приоритетные чек-листы после рождения
3. остальные чек-листы
4. последние события

## События
### `PREPARING`
Видны:
- подготовительные действия
- заметка этапа
- вход в `Активные роды`

### `LABOR`
Видны:
- start или stop схватки
- воды
- рождение
- логистика
- заметка этапа

### `BABY_BORN`
Видны:
- действия после рождения
- карточка ребенка
- бытовые записи
- заметка этапа

`Начались схватки` и live labor tools не являются основными в `BABY_BORN`.

## Активные роды
Экран показывает:
- рекомендацию
- текущую схватку и интервал
- статус вод
- историю схваток
- действия start или stop, water break, birth

## Чек-листы
- фильтр по умолчанию равен текущему этапу
- пользователь может выбрать другой этап вручную
- экран показывает и системные, и пользовательские чек-листы
- удаленный системный чек-лист больше не возвращается автоматически

## Журнал
Показываются:
- системные события
- пользовательские заметки

Фильтр `Мои заметки` использует `entryType = USER_NOTE`, а не `TimelineType`.
